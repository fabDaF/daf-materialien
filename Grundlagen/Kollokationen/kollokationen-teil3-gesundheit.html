In den ersten beiden Lektionen haben wir Kollokationen in den Bereichen Leistung und Meinung geübt. Jetzt geht es um ein Thema, das uns alle betrifft: Gesundheit, Krankheit und Behandlung.

Ob beim Arztbesuch, in den Nachrichten oder in wissenschaftlichen Texten – wer über Medizin spricht, braucht präzise Wortkombinationen. Eine Krankheit haben versteht jeder, aber natürlich klingt nur an einer Krankheit leiden.

Was macht den Unterschied?

Auch im Bereich Gesundheit sehen wir drei wichtige Muster:

1. Das Verb – präziser statt banal

Die Allerweltsverben sagen, machen, bekommen, geben und anfangen weichen den medizinisch präzisen Verben.

eine Diagnose sagen → eine Diagnose stellen
eine Behandlung bekommen → sich einer Behandlung unterziehen
eine Operation machen → eine Operation durchführen
eine Therapie anfangen → eine Therapie einleiten
eine Krankheit kaputtmachen → eine Krankheit bekämpfen
eine Verletzung bekommen → eine Verletzung erleiden
Schmerz weniger machen → Schmerz lindern
einen Eingriff machen → einen Eingriff vornehmen
ein Medikament geben → ein Medikament verschreiben

2. Das Adjektiv – treffender statt vage

Statt gut, schlimm, lang und groß verwenden wir Adjektive, die in der Medizinsprache fest mit dem Nomen verbunden sind.

eine lange Behandlung → eine langwierige Behandlung
eine gute Therapie → eine gezielte Therapie
eine schlimme Verletzung → eine schwere Verletzung
starker Schmerz → heftiger Schmerz
ein gutes Medikament → ein wirksames Medikament
ein kleiner Eingriff → ein chirurgischer Eingriff

3. Die Kette – ein Nomen, viele Phasen

Aber es gibt noch mehr: Jedes Nomen trägt ein ganzes Universum an Verben mit sich. Diese Verben beschreiben verschiedene Phasen – wie eine Geschichte, die sich entwickelt.

die Krankheit: diagnostizieren → behandeln → bekämpfen → besiegen
(Erkennung → Therapie → Kampf → Heilung)

der Schmerz: verursachen → empfinden → lindern → überwinden
(Ursache → Erleben → Hilfe → Ende)

das Medikament: verschreiben → einnehmen → absetzen
(Verordnung → Einnahme → Ende)

Jedes dieser Verben ist eine eigene Kollokation – und zusammen bilden sie eine Kollokationskette. In Tab 4 übst du diese Ketten.

Was bedeutet das für dich?

Im medizinischen Bereich zählt Präzision besonders. Wer eine Diagnose nicht sagt, sondern stellt, und eine Verletzung nicht bekommt, sondern erleidet, klingt nicht nur kompetenter – er wird auch besser verstanden.

Und denk daran: Jedes Nomen hat nicht nur ein passendes Verb, sondern eine ganze Kette. Wer diese Ketten kennt, beherrscht das Thema wirklich.
