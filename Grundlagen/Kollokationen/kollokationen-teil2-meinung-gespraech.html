In der ersten Lektion haben wir gesehen, wie Kollokationen im Bereich Leistung und Bewertung funktionieren. Jetzt geht es um einen Bereich, in dem sie genauso wichtig sind: Meinung, Gespräch und Überzeugung.

Wer über Meinungen spricht, Argumente austauscht oder Gespräche beschreibt, braucht präzise Wortkombinationen. Wieder gilt: Grammatisch ist vieles möglich – aber natürlich klingt nur die richtige Kollokation.

Vergleiche die folgenden Satzpaare. Links steht die banale Version, rechts die natürliche.

Was macht den Unterschied?

Auch in diesem Themenfeld sehen wir drei wichtige Muster:

1. Das Verb – präziser statt banal

Wieder ersetzen wir die Allerweltsverben sagen, machen, haben, bekommen durch Verben, die genau zum Nomen passen.

eine Meinung sagen → eine Meinung vertreten
ein Argument sagen → ein Argument vorbringen
eine Frage sagen → eine Frage aufwerfen
eine Antwort sagen → eine Antwort erteilen
ein Gespräch machen → ein Gespräch führen
ein Versprechen machen → ein Versprechen abgeben
Vertrauen bekommen → Vertrauen gewinnen
Zweifel haben → Zweifel hegen
Einfluss haben → Einfluss ausüben

2. Das Adjektiv – treffender statt vage

Statt gut, groß und wichtig verwenden wir Adjektive, die zum jeweiligen Nomen gehören wie ein Schlüssel zum Schloss.

ein gutes Argument → ein überzeugendes Argument
eine wichtige Frage → eine entscheidende Frage
eine gute Antwort → eine zufriedenstellende Antwort
ein langes Gespräch → ein ausführliches Gespräch
großer Einfluss → erheblicher Einfluss
Zweifel → ernsthafte Zweifel

3. Die Kette – ein Nomen, viele Phasen

Aber es gibt noch mehr: Jedes Nomen trägt ein ganzes Universum an Verben mit sich. Diese Verben beschreiben verschiedene Phasen – wie eine Geschichte, die sich entwickelt.

das Versprechen: abgeben → einhalten → brechen
(Anfang → Treue → Scheitern)

das Vertrauen: aufbauen → gewinnen → erschüttern → verlieren
(Aufbau → Erfolg → Krise → Verlust)

die Meinung: bilden → vertreten → ändern
(Entstehung → Aktivität → Wandel)

Jedes dieser Verben ist eine eigene Kollokation – und zusammen bilden sie eine Kollokationskette. In Tab 4 übst du diese Ketten.

Was bedeutet das für dich?

In Diskussionen, Präsentationen und schriftlichen Texten zählt jedes Wort. Wer präzise Kollokationen benutzt, wirkt nicht nur sprachlich kompetent – er überzeugt auch besser.

Denk daran: Ein Argument wird nicht gesagt, sondern vorgebracht. Vertrauen wird nicht bekommen, sondern gewonnen. Und Zweifel werden nicht gehabt, sondern gehegt.

Und denk daran: Jedes Nomen hat nicht nur ein passendes Verb, sondern eine ganze Kette. Wer diese Ketten kennt, beherrscht das Thema wirklich.
