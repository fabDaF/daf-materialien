📝 Was? / Wohin?

Wechselpräpositionen mit „auf" + Akkusativ (Wohin?)

  • Wohin? → Akkusativ: auf den Tisch, auf die Bank, auf das Sofa
  • auf + das = aufs (informell: „aufs Sofa")
  • Bewegung = Akkusativ (wohin geht/fährt/steigt etwas?)

Wechselpräpositionen mit „auf" + Akkusativ (Was? / Wohin?)

  • Imperativ-Sätze: Stell/Leg/Setz + Akkusativ-Objekt + auf + Akkusativ-Ort
  • auf + den Tisch, auf + die Couch, auf + das Bett
  • Artikel im Akkusativ: den/die/das (maskulin/feminin/neutral)
